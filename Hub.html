Welcome to Samba

Samba is an early minimalist Web browser from 1992, minimalist browser displayed only text, no images nor lists.It was the first web browser for the classic Mac OS platform,tries to emulate the design of WWW. It was written at CERN “The European Organization for Nuclear Research” by Robert Cailliau . It was Implemented in THINK C using its human interface objects

Web Design Vs. Web Development

web designer Focuses on look and feel of a website Uses HTML, CSS, and JavaScript Is competent in color scheming, graphic design, and information flow Creates a great user experience Is right-brained: Strong intuition, creativity, & imagination Portfolio is worth more than a degree,Median salary $47,820,

web developer Creates the inner workings of a website Is competent in programming languages (PHP, .NET, Python, C, Ruby) Develops the user interface Is left-brained: logic, linear thinking, technical Hands-on coding experience is required (either your own products or github) Median salary: $85,430
